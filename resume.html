2015.10.1-至今 AndroidUI-WebApp 框架

将 Android SDK 的 UI 层 ( view, widget, graphics 等相关包) 移植至 javascript 实现，绘制层(canvas)使用平台自己的绘制接口(web canvas / android canvas)，以此做到由 js 驱动 UI 的展示和交互。
特点：
跨平台&动态性：js驱动，一份代码多个平台(web,android,ios)。
性能：主流手机有50～60fps流畅度。
移植的组件：安卓的UI组件(官方&第三方)都可以使用。
易上手：对Android开发来说只需学习js语言语法。
App&Web: WebApp, Hybrid App, App全支持。
